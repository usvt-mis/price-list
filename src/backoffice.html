What Saves Here

The settings in this tab are shared across users: landing page card visibility, motor job defaults, branch markup percentages, and Sales Quotes print layout.

Scheduling Tips

  • Leave both date fields empty to keep a card available continuously.
  • Set only a start date to delay opening until that moment.
  • Set only an end date to close the card after that moment.
  • Time Board still respects role access even if you enable its card schedule.

Motor Default Tips

  • Use short prefixes like wire arc when several jobs share the same family.
  • Leave the list empty if you want every job to load as checked by default.
  • Changes here affect future loads only. Users can still manually check or uncheck jobs later.

Print Layout Tips

  • Save small changes in batches, then verify from the print preview.
  • Use Use Defaults if the shared layout drifts too far and you need a clean baseline.
  • Every save here affects all Sales Quotes users.
